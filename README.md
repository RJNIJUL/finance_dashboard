# 💰 Finance Dashboard

A full-stack personal finance dashboard built using Flask.

## Features
- User authentication (login/register)
- Add, edit, delete transactions
- Budget tracking
- Spending analytics (charts)
- Profile with image upload
- Activity tracking
- Dark mode

## Tech Stack
- Python (Flask)
- SQLite
- HTML, CSS, JavaScript
- Bootstrap

## How to Run
1. Install requirements:
   pip install flask

2. Run app:
   python app.py

3. Open:
   http://127.0.0.1:5000
